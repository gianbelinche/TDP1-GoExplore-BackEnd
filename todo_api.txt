POST /users
GET /users/{id} -> Informacion publica de usuarios (incluye tarjetas)
POST /session -> Crea la sesion
POST /user/{id}/card
POST /experiences
GET /experiences -> Busca las experiencias
  - lat
  - lng
  - distance
  - user_id
  - category

GET /experiences/{id} -> Trae la experiencia de id
POST /images -> Sube una lista de imagenes
GET /images/{id} -> Trae la imagen de id

POST /booking -> Crea la reserva
por ver:
GET /bookings/mine
GET /users/{id}/bookings
